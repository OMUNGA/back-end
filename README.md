## Description

Easy to use to be able to write your content

## Installation

```bash
$ npm install or yarn install
```

## Running the app with docker
  ```bash
# development
$ docker-compose up
```


## Running the app 

```bash

# development
$ npm run dev

# or
$ yarn dev
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```

## Running the migrations
  ```bash
# development
$ yarn migration:run

```

## Create migration
  ```bash
# development
$ yarn migration:create --name=name_migration

```

## Revert Migration
  ```bash
# development
$ yarn migration:revert

```
